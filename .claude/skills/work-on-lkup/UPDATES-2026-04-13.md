## Enrichment Architecture (3-Method Parallel)

All 7 grading services fire up to 3 methods in parallel:
- PCGS: API → E2B
- NGC: Direct HTTP → CF Browser → E2B
- ICG: Direct HTTP → CF Interactive
- CAC: CF Browser → E2B
- ANACS: CF Interactive → E2B
- SEGS: CF Interactive → E2B
- Gongbo: CF Browser
